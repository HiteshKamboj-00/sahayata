## 📖 About

*Sahayata* is a web-based assistance platform designed to connect people with the help they need. Whether it's community support, resources, or guidance — Sahayata makes it accessible and simple.

---

## ✨ Features

- 🖥️ Clean, modern user interface
- 📱 Fully responsive design (mobile-friendly)
- ⚡ Lightweight — no frameworks or dependencies
- 🎨 Custom CSS styling
- 🔧 Interactive elements powered by vanilla JavaScript

## 🚀 Getting Started

### Prerequisites

No installations required! This is a pure frontend project — just a browser is all you need.

### Run Locally

1. *Clone the repository*
   bash
   git clone https://github.com/HiteshKamboj-00/sahayata.git
   

2. *Navigate into the project folder*
   bash
   cd sahayata
   

3. *Open in your browser*
   bash
   open index.html
   
   Or simply double-click index.html in your file explorer.

---

## 💻 Usage

Once opened in a browser:
1. Navigate through the interface to explore available help resources.
2. Use the interactive elements to find assistance or submit a query.
3. The site is fully responsive — works great on mobile and desktop.
